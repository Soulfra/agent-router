Unlike 99% of websites, we don't use tracking cookies, Google Analytics, Facebook Pixel, or any third-party analytics. We respect your privacy.

What Data We Collect

1. Account Information (When You Sign Up)

  • Email address - For login and account recovery
  • Username/handle - Your chosen display name
  • Password - Encrypted with bcrypt (we can't see it)

2. Session Data (Temporary)

  • Session ID - Random token to keep you logged in
  • Device fingerprint - SHA-256 hash for security (not tracking)
  • IP address - For location-based features and security
  • User agent - Browser/device info for compatibility

Note: Session data is deleted when you log out or after 30 days of inactivity.

3. Usage Data (Anonymous)

  • Page views - Which pages are popular (no individual tracking)
  • Feature usage - Which features get used (aggregated)
  • Error logs - To fix bugs (no personal info)

4. Payment Data (If You Purchase)

  • Stripe handles payment - We never see your card number
  • We store - Order ID, amount, timestamp
  • We don't store - Credit card numbers, CVV, billing address

How We Track Attribution (Without Cookies)

Instead of cookies, we use privacy-friendly methods:

  • QR Codes - Scan a QR code with a unique session token
  • Referral Links - URLs with affiliate codes (e.g., ?ref=PARTNER-CODE)
  • Device Fingerprinting - Anonymous hash of device characteristics (for security, not tracking)
  • Server-side Sessions - Session ID stored on our server (not in browser cookies)

Example: If an affiliate shares a referral link (https://calos.ai?ref=PARTNER123), we track conversions by that code, not by following you around with cookies.

Your Rights

1. Access Your Data

Email us at privacy@calos.ai to request a copy of your data.

2. Delete Your Data

Go to Account Settings ‚Üí Delete Account. All your data will be permanently deleted within 30 days.

3. Opt Out of Analytics

Analytics are already minimal and anonymous. If you want complete privacy, use a VPN or Tor Browser.

4. Portability

Export your data anytime from Account Settings ‚Üí Export Data (JSON format).

Data Retention

  • Active accounts - Data kept indefinitely
  • Inactive accounts (1 year+) - Reminder email, then deletion after 90 days
  • Deleted accounts - 30-day grace period, then permanent deletion
  • Session data - Deleted after 30 days
  • Logs - Deleted after 90 days

Security

  • Encryption - All data encrypted at rest (AES-256) and in transit (TLS 1.3)
  • Passwords - Hashed with bcrypt (can't be reversed)
  • API keys - Stored with SHA-256 hashes
  • Two-factor authentication - Optional, recommended

Third-Party Services

We use these services, but they don't track you:

  • Stripe - Payment processing (PCI-DSS compliant)
  • AWS/Cloudflare - Hosting and CDN (no tracking)
  • PostgreSQL - Database (self-hosted)

We do NOT use:

  • Google Analytics
  • Facebook Pixel
  • Google Ads
  • Any ad network
  • Any data brokers

Children's Privacy

CALOS is not intended for children under 13. We don't knowingly collect data from children. If you're a parent and believe your child has created an account, contact us at privacy@calos.ai.

Changes to This Policy

If we update this policy, we'll notify you via email and display a banner on the site. Continued use after changes means you accept the new policy.
