How BYOK Works

Your API keys are encrypted with AES-256-GCM and stored securely. CalOS uses a contextual key retrieval system:

  • Tenant BYOK → Your organization's keys (highest priority)
  • User Keys → Your personal API keys
  • System Keys → Fallback to platform keys (charges apply)
